In this we can learn basic command of Git and Github.
First we learn the terminology behind it (Source code management system).
There are mainly two type of Source code management-
1.CVCS-Centralize version control system
  In simple it is not locally availabale.We always need to be connected to a network to perform any action.
  Hence everything is centralize if central sevrer gets failed you lost entire data.
2.DVCS-Distributed version control system
  In DVCS evryone(contributer) has a local copy of main repository.Hence evryone maintained a local repository of their own.
